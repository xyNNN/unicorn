.. title:: DataTransferConverter

=====================
DataTransferConverter
=====================

Use the ``DataTransferConverter`` to convert between different temperature units, f.e. from Megabyte to Gigabyte.

Base Unit
=========

- Celsius

Features
========

- Conversion
- Mathematical operations
- Adding own units

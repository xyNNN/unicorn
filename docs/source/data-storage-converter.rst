.. title:: DataStorageConverter

====================
DataStorageConverter
====================

Use the ``DataStorageConverter`` to convert between different temperature units, f.e. from Megabyte to Gigabyte.

Base Unit
=========

- Celsius

Features
========

- Conversion
- Mathematical operations
- Adding own units
